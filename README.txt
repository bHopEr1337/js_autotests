# Автоматизация тестирования веб-сайта ИнфоТеКС с использованием Playwright

Данный проект представляет собой автотесты на JavaScript с использованием библиотеки playwright. Основной целью проекта является верификация функциональности сайта и проверка отображения информации в различных разделах.

## Цели проекта

В рамках проекта реализованы автоматизированные тесты для выполнения следующих задач:
- Проверка наличия необходимых вкладок в разделе "О компании".
- Сравнение количества патентов с использованием фильтров.
- Поиск и подсчет всех патентов на главной странице.

## Задания

### 1. Проверка раздела "О компании"
Реализован тест, который осуществляет переход на сайт ИнфоТеКС и открывает раздел "О компании". Тест проверяет наличие следующих вкладок:
- Компания «ИнфоТеКС»
- Экосистема ИнфоТеКС
- Лицензии
- Академия
- Патенты
- Награды
- Реквизиты
- Вакансии
- Контакты

### 2. Анализ раздела "Патенты"
Разработан тест, который переходит в раздел "Патенты" и использует фильтр "Патенты РФ на изобретения". Цель теста заключается в сравнении количества патентов, отображаемых на странице.

### 3. Поиск патентов
Создан тест, который использует поле поиска на главной странице для нахождения всех патентов и вывода их количества в консоль.

## Общие требования

Для обеспечения стабильности и надежности тестов соблюдены следующие требования:
- Использование локаторов на основе data атрибутов и CSS свойств.
- Разработка проекта по шаблону POM (Page Object Model) для улучшения структуры и поддерживаемости кода.
- Формирование отчета по результатам тестирования в формате HTML.

## Установка и запуск проекта

Для запуска проекта необходимо выполнить следующие шаги:

1. Распаковать архив с проектом и открыть его в удобной IDE (например, Visual Studio Code).

2. В корневом каталоге проекта следует выполнить команды:

   cd js-autotest
   npm install playwright
   npx playwright install

3. Для запуска тестов используется команда:

   npx playwright test

После завершения выполнения тестов браузер автоматически откроет отчет index.html, содержащий ссылки на отчеты по каждому из тестов.
